**Struktur:**
```
Empfohlene Sektionen:
1. Hero Section (Above the Fold)
2. Social Proof / Client Logos
3. Problem-Solution Statement
4. Key Features/Benefits
5. How It Works / Product Demo
6. Case Studies / Success Stories
7. Pricing Preview (optional)
8. FAQ Section
9. Final CTA Section
10. Footer
```

**Umfang:**
- Seitenlänge: Multi-Page Website
- Detailgrad: Vollständiges Design-Konzept
- Anzahl der Iterationen: 3-4

**Enthalten sein muss:**
- Klare Value Proposition
- Trust Signals (Zertifikate, Kundenstimmen, Logos)
- Mindestens ein starker Call-to-Action
- Mobile Responsive Überlegungen
- SEO-freundliche Struktur
- Conversion-fokussierte Elemente

**Nicht enthalten:**
- Finale grafische Designs, Stock-Fotos, vollständige Programmierung - dies erfolgt separat

## 8. EINSCHRÄNKUNGEN & VORGABEN

**NICHT tun:**
- Keine übertriebenen Versprechungen oder Superlative ohne Beleg
- Keine generischen Stock-Foto-Empfehlungen ohne Kontext
- Keine komplizierten technischen Details im Above-the-Fold Bereich
- Keine Verletzung von Markenrichtlinien falls vorhanden
- Keine Navigation die von der Conversion ablenkt
- Keine Auto-Play Videos oder störende Animationen

**MUSS enthalten:**
- DSGVO-konforme Datenschutzhinweise bei Formularen
- Barrierefreiheit (WCAG 2.1 AA Mindeststandard)
- Klare Kontaktmöglichkeiten
- Transparente Informationen über das Produkt
- Mobile-optimierte Darstellung
- Schnelle Ladezeiten-Optimierung

**Präferenzen:**
- Fokus auf Conversion-Optimierung
- Datengetriebene Design-Entscheidungen
- A/B-Test-freundliche Varianten
- Modularer Aufbau für einfache Anpassungen
- Klar strukturierte Content-Hierarchie
